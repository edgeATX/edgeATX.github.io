"React-like"
View Libraries

React Changed the Game

React is fast, focused and well designed.

New view library trend: "React-like", but...

  • Smaller
  • Faster
  • Simpler
  • All of the above?

Vue.js

Reactive Components for Modern Web Interfaces
  • Composable views + reactive data bindings
  • Goal: simplest possible API
  • Compare Vue to other frameworks

Riot

A React-like user interface micro-library
  • Minimal, "enjoyable syntax" (9kb zipped)
  • Reactive views, routing, events
  • Custom tags (similar to web components)

Preact

Fast 3kb alternative to React with the same ES6 API
  • React's "core value proposition" with as little code as possible
  • Includes most of React's API
  • Extra conveniences (auto-passing of state/props, class attribute supported, etc.)

Inferno

An extremely fast, React-like JS library for building modern user interfaces
  • Distinguishes static / dynamic content for fast rendering
  • Replicates the "best features from React"
  • Again, it has a few differences

In Case You
Missed It

Hyperdev

HyperDev is the developer playground for building real web apps, fast

Joel announced the beta May 31

  • Instant hosting, automated deployment, no setup
  • Collaborative real-time editing, instant save
  • Share and remix projects

Cutestrap

A sassy, opinionated CSS Framework. A tiny alternative to Bootstrap.
  • Only 8kb of CSS
  • Forms, buttons, grid & typography

is.js

Minimalistic predicate library
  • Unopinionated, zero dependencies
  • Works with Node, AMD and all browsers (IE6+)
is.array([]) // true
is.not.integer(0) // false
is.propertyDefined({foo: {bar: 0}} , 'foo.bar') // true
is.deepEqual([1 , [2, 3]], [1, [2, 3]]) // true
