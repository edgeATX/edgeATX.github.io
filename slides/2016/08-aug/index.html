Opera 39

Released June 8 — Release Notes

  • Added support for Chromium version 51
  • New Power Saver mode
  • Mostly new end user features

Firefox 48

Released August 2 — Release Notes

  • Improved CSS grid support, now works in fieldset
  • New ES2015 methods: Array.prototype.values, String.prototype.padStart / padEnd and more
  • String.prototype.contains replaced with String.prototype.includes
  • Minor improvements to Web Workers, WebRTC, Dev Tools and more
  • Lots more minor HTML, CSS and JS improvements

Edge 14 (14367)

Release Notes

  • Mostly performance and bug fixes

Bonus: Safari Technology Preview 11

Released August 17 — Release Notes

  • 11th preview release since March
  • Minor fixes and improvements (polishing stage)
  • Already at 100% ES2015 compatibility!
  • Download to preview Safari 10
