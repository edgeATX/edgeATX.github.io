Chrome Highlights

Version 43

Enabled by default:
  • Unprefixed CSS Animations (no -webkit).
  • Cache API (global scope).
  • Fetch API: streaming response body. (produces ReadableStream)
  • Permissions API
  • autocapitalize.
<input type="text" autocapitalize="sentences"> <input type="text" autocapitalize="words"> <input type="text" autocapitalize="characters">

Firefox Highlights

Version 38

  • fetch() now implemented.
  • Websockets can be used in workers.
  • Better ES6 support for standard features (generators, maps, weakmaps)
  • Array methods for typed arrays: of() from() forEach() filter() map() slice()
  • <picture> now activated by default.
  • Devtools: Identify XHR Requests
