Chrome Highlights

Version 45 Beta

Notable ES6 features:

  • Arrow functions (=>)
  • New Array and TypedArray methods (Array.prototype.find() — yay!)
  • Object.assign()

Plus service worker improvements and a bunch of other stuff

More details next month :)

Firefox Highlights

Version 40

  • Lots of dev tools improvements, as usual
  • Experimental Push API support
  • Mostly minor ES6 additions and fixes

Opera

Version 31

  • Updated Chromium/Blink to version 44
  • New CSS multi-column layout implementation

Bootstrap 4

Bootstrap 4 has been a massive undertaking that touches nearly every line of code.
  • Dropped IE8 support
  • Less → Sass
  • Improved grid system + flexbox support
  • Wells / thumbnails / panels → cards
  • Every plugin now written in ES6
  • Lots, lots more
