Trends & Predictions

Trend: Parallel/Delegated Processing

  • WebRTC
  • Shared Memory & Atomics

  • Cooperative Scheduling of Background Tasks API (Queues)
  • Budget API
  • Network/Bandwidth Controls

Prediction: Fat Clients & Servers

  • Hardware is on steroids.
  • Software is on steroids.
  • Instant gratification is a norm.

Computing has historically oscillated between thin client/fat server and fat client/thin server models. Hardware and software have piggybacked off each other to create this effect. Both continue evolving their power, creating a fat client/fat server model.

Prediction: Labor Split | Confusion

  • UI Designers
  • UX Researchers
  • UX Developers
  • UI Developers
  • UI Engineers

Traditonal "computer science", "networking", and "data management" continue to breach the "great frontend barrier". Greater knowledge will be necessary to adequately use new features, ultimately leading to mental overload. To compensate, the market will split job duties by vertical knowledge silos within organizations.
