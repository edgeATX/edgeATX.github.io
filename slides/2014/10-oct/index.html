Chrome Highlights

Version 38

  • Responsive images via <picture>, <source> and the srcset attribute
  • Support for ES6 Maps, Sets and iterable objects
  • A bunch of new ES6 math functions
  • Support for W3C Screen Orientation API
  • Over $75k in security bounties fixed

Firefox Highlights

Version 32

  • New Web Audio Editor (still beta)
  • ES6 (experimental) methods: Array.from(), Array.prototype.copyWithin(), Number.isSafeInteger()
  • CSS properties: mix-blend-mode and position: sticky
  • Flexbox properties: flex-grow and flex-shrink (candidate recommendation W3C spec)
  • Experimental support: <img> srcset, box-decoration-break

Firefox Highlights

Version 33

  • New CSS property: @counter-style
  • A ton of dev tools additions:
    • Event listeners popup
    • @media sidebar
    • CSS transform highlighting
    • more...

</div>

<style>
  .shapes-example,
  .shapes .shapes-preview {
    display: none;
  }

    display: block;
  }
</style>

Built-in shapes: circle, ellipse, polygon

CSS Shapes - Level 1

Why is this interesting?

  • Finally able to create print-quality layouts natively in HTML
  • W3C Candidate Recommendation status
  • Now supported by default in Chrome, Opera and Safari *
  • Polyfill available for other browsers
  • WSYWIG editors available for Chrome and Brackets

* Surprisingly, Firefox typically leads on W3C spec support, but does NOT currently support Shapes.

CSS Shapes - Level 1

Why "Level 1"?

There is also a Level 2 spec (currently in Editor's Draft status)

Will extend Level 1:

  • Adds support for defining "exclusion" areas via shape-inside
  • Support for standard positioning, in addition to floats

Polyfill.io

https://cdn.polyfill.io/v1/ (Overview)

Hosted HTML5 / CSS3 / ES6 polyfill delivery service

A way of sending one single copy of each of the best polyfills, in the right order (respecting dependencies), to just the browsers that lack a native implementation.
  • Reads the requesting browser's UA string (customized per request)
  • Eliminates duplication and manual polyfill management
  • "Care-free" ability to leverage latest standards
