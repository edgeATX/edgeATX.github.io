Tips & Tricks

Designing for Dark Mode

Implementing dark mode is easy, but designing for it is less so.
  • OSX is pushing dark mode with Mojave
  • Here is a great overview of things to consider
  • Choosing your "dark" color, typography, consistency

Designing for Dark Mode

This is an emerging Level 5 Media Query standard:

@media (prefers-color-scheme: dark) {
  /* dark mode styles */
}
  • Possible values: light, dark, or no-preference
  • Supported today in Safari 12.1
  • Slated for Chrome 76 & Firefox 67
  • Here's a React-specific tutorial
